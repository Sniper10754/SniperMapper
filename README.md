# SniperMapper [![](https://jitpack.io/v/Sniper10754/SniperMapper.svg)](https://jitpack.io/#Sniper10754/SniperMapper)

### Example

```java
TCPScanner scanner = new TCPScanner("address");

int[] openports = scanner.scanPorts(0, 80);
```
